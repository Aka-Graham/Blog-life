### 有些git命令还经常遇到的，但是总是想不起来。

#### 修改本地分支名称
git branch -m old new 

example: git branch -m master main

#### 修改远程分支名称
1. 删除远程分支
git push --delete origin oldBranch
2. 提交本地分支

#### 配置推送的仓库和分支，连接本地分支和远程分支

git branch --set-upstream-to=origin/main main

#### 删除远程仓库

git remote rm <远程仓库名称(origin)>
